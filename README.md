# UTP-Study-Spot-Reccomender-AI-Project-
A web-based intelligent system that uses Fuzzy Logic concepts to recommend the ideal study location on the UTP campus based on a student's specific preferences for quietness, crowd levels, and power outlet availability. Simply double-click index.html to open it in your web browser. No local server or backend is required.
